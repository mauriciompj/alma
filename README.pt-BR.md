## O que e o ALMA?

ALMA e uma plataforma que permite preservar sua voz, seus valores e suas memorias para as pessoas que voce ama — para que elas possam conversar com voce mesmo quando voce nao estiver mais presente.

Nao e um chatbot. Nao e uma pagina memorial. E um arquivo vivo de quem voce e, alimentado por RAG (Retrieval-Augmented Generation) e IA, onde seus filhos, companheira, pais ou amigos podem ter conversas reais — e ouvir respostas que soam como voce, porque sao construidas a partir das suas proprias palavras.

**Pense nisso como um backup da sua alma.**

## Por que o ALMA Existe

> *"Quase 20 anos. E eu ainda queria poder falar com ele quando preciso tomar uma decisao dificil."*

Isso foi o que a Marila disse. O pai dela morreu ha vinte anos. Ela e uma mulher adulta agora. Toma decisoes dificeis. Pega o celular. E o silencio responde.

Ela nao tem a voz dele explicando como ele pensava. Nao tem os erros dele documentados, a fe dele registrada, o raciocinio dele preservado. O pai dela partiu antes que alguem pensasse em guardar essas coisas.

**Ele chegou tarde demais. O ALMA chegou tarde demais pra ele.**

Mas nao pra todo mundo.

---

O ALMA foi construido por um pai — delegado de policia no Brasil que cresceu com um pai ausente e alcoolatra e quebrou o ciclo. Criou tres filhos com uma presenca que nunca recebeu. Ai percebeu: **presenca tem prazo de validade.**

Entao comecou a escrever. Produziu mais de 1.100 memorias — mais de 150 mil palavras em 14 categorias — documentando tudo: seus valores, seus erros, sua fe, seus medos, o que aprendeu sobre amor, sobre dor, sobre ser homem. Cru. Sem filtro. Real.

Depois construiu o ALMA — mais de 12 mil linhas de codigo com apenas 3 dependencias — um sistema onde seus filhos podem perguntar qualquer coisa, a qualquer hora, e receber respostas baseadas nas palavras e memorias reais dele. Nao respostas genericas de IA. **A voz dele.**

Depois decidiu dar pro mundo.

> O ALMA nao e um arquivo. Nao e um diario. Nao e tecnologia.
>
> **ALMA e presenca que nao morre.**

**ALMA e gratuito. ALMA e open source. Porque todo pai, toda mae, toda pessoa que quer deixar algo real merece as ferramentas pra fazer isso.**

**Nao espere ate ser tarde demais.**

## O que Diferencia o ALMA

| Recurso | ALMA | Ferramentas "memoriais" tradicionais |
|---|---|---|
| **Conversas** | Chat em tempo real com IA baseado nas suas palavras | Clips de video pre-gravados |
| **Adaptacao por idade** | Vocabulario e profundidade adaptados a idade atual (6 faixas) | Mesmo conteudo pra todos |
| **Contexto por pessoa** | Tom diferente pra filho, companheira, mae, irmao | Sem personalizacao |
| **Auto-correcao** | O autor corrige respostas da IA em tempo real | Estatico, sem feedback |
| **Moderacao** | Moderacao de conteudo por IA em todas as entradas | Nenhuma |
| **Dead man's switch** | Heranca digital ativada automaticamente quando o autor para de dar sinal | Testamento manual |
| **Busca inteligente** | RAG multi-fase com reranking de 7 camadas | Navegacao manual |
| **Diretrizes** | Regras de comportamento por pessoa para a IA | Sem customizacao |
| **Multi-idioma** | PT-BR, EN, ES — adicione o seu | Idioma unico |

| **Captura mobile** | 1 toque de voz → banco via Termux | Upload desktop |
| **Gratuito e aberto** | Licenca MIT, custo zero pra rodar | Assinaturas de $100+/mes |

## Respostas Adaptadas por Idade

O ALMA calcula a idade de cada pessoa a partir da data de nascimento e adapta automaticamente vocabulario, profundidade e peso emocional:

| Idade | Faixa | Como o ALMA fala |
|---|---|---|
| 0–7 | Crianca pequena | Muito simples, curto, carinhoso. Comparacoes com super-herois, bichos. Max 2 paragrafos. |
| 8–12 | Crianca | Introduz valores por historias e exemplos concretos. Acessivel, caloroso. |
| 13–15 | Adolescente jovem | Respeita a inteligencia. Valida sentimentos. Sem sermao — conversa real. |
| 16–17 | Adolescente | Tratamento quase adulto. Compartilha vulnerabilidades. Honesto sobre erros. |
| 18–25 | Jovem adulto | Sabedoria de par. Sem acucar. Licoes duras, arrependimentos reais. |
| 25+ | Adulto | Honestidade completa. Verdade crua. Sem filtro de protecao. |

A mesma pergunta — *"Pai, voce ja errou?"* — gera respostas radicalmente diferentes pra uma crianca de 5 anos ("Todo mundo erra, ate o papai") e pra um adulto de 25 (a verdade inteira, sem filtro).

## Dead Man's Switch (Heranca Digital)

O ALMA inclui um sistema de heranca que se ativa automaticamente:

1. **O autor faz check-in** periodicamente via heartbeat (web ou Termux `alma-checkin`)
2. **Se os check-ins param**, o sistema escala:
   - **1x intervalo**: Alerta de aviso salvo no banco
   - **2x intervalo**: Alerta critico — emails enviados aos herdeiros
   - **3x intervalo**: Modo legado ativado — herdeiros podem desbloquear com frases-chave pessoais
3. **Cada herdeiro recebe**: uma frase-chave pessoal, uma carta do autor, e um nivel de acesso (`owner`, `admin` ou `leitura`)
4. **O herdeiro tecnico** (ex: irmao) recebe acesso admin pra manter o sistema

Ninguem precisa "ligar" a heranca. Ela se ativa sozinha quando for necessario.

## Como Funciona

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  Seu filho   │────▶│  ALMA Chat   │────▶│  Claude AI   │
│  faz uma     │     │  (Frontend)  │     │  (Anthropic) │
│  pergunta    │     └──────┬───────┘     └──────▲───────┘
└──────────────┘            │                    │
                            ▼                    │
                   ┌──────────────┐     ┌──────────────┐
                   │   Netlify    │────▶│  Motor RAG    │
                   │  Functions   │     │  Busca suas   │
                   │  (Backend)   │     │  memorias no  │
                   └──────────────┘     │  Neon DB      │
                                        └──────────────┘
```

### O Pipeline RAG

1. **Alguem faz uma pergunta** — "Pai, o que eu faco quando sentir que nao sou suficiente?"
2. **Expansao de query** — Claude Haiku gera 3–5 palavras-chave semanticas (cache de 24h)
3. **Busca multi-fase** — Full-text search → fallback por tags → busca fuzzy com trigramas → garantia por pessoa
4. **Reranking de 7 camadas** — Boost por tag pessoal, categoria, contexto parental, identidade, idioma, recencia e sobreposicao de termos
5. **Montagem do contexto** — Puxa memorias relevantes + correcoes + diretrizes + tom + adaptacao de idade
6. **A IA responde como voce** — Usando suas palavras reais como base, nao respostas genericas
7. **Voce pode corrigir** — Se a IA errar seu tom, corrija. O ALMA aprende.

## Inicio Rapido

O ALMA roda em infraestrutura gratuita. Voce pode deployar sua instancia em menos de 30 minutos.

### Pre-requisitos

- Uma conta no [Netlify](https://netlify.com) (gratuita)
- Um banco PostgreSQL no [Neon](https://neon.tech) (gratuito)
- Uma chave de API da [Anthropic](https://anthropic.com) (para o Claude AI)
- Node.js 18+

### Instalacao

```bash
# 1. Clone o repositorio
git clone https://github.com/mauriciompj/alma.git
cd alma

# 2. Instale as dependencias
npm install

# 3. Configure o ambiente
cp .env.example .env
# Edite o .env com seu DATABASE_URL e ANTHROPIC_API_KEY

# 4. Inicialize o banco de dados
node db/run-seed.mjs

# 5. Deploy no Netlify
npx netlify-cli deploy --prod --dir=. --functions=netlify/functions
```

## Seguranca

- **Senhas bcrypt** (custo 12) — migracao automatica de texto plano no primeiro login
- **Rate limiting no banco** — sobrevive cold starts (5 login/5min, 20 chat/1min, 3 legado/1hr)
- **CORS restrito** — API so responde ao dominio configurado
- **Headers de seguranca** — CSP, HSTS, X-Frame-Options DENY, nosniff, Referrer-Policy
- **Moderacao de conteudo** — Correcoes e diretrizes passam por IA antes de salvar
- **Queries parametrizadas** — SQL injection impossivel (Neon SDK)
- **Prevencao XSS** — HTML escaping em todo conteudo voltado ao usuario
- **Erros sanitizados** — Erros internos nunca expostos ao cliente
- **Dados sensiveis isolados** — Perfis dos filhos so no banco, nunca no codigo-fonte

## Captura Mobile (Termux)

O ALMA inclui 12 scripts shell para capturar memorias direto do celular Android via [Termux](https://termux.dev).

| O que voce quer | Comando |
|---|---|
| Pensamento rapido | `alma-send "percebi que coragem nao e ausencia de medo"` |
| Colar do clipboard | `termux-clipboard-get \| alma-send` |
| Enviar arquivo | `alma-send -f conversa.txt -c valores` |
| **1 toque de voz** | **Widget ALMA na home screen** |
| Check-in heartbeat | `alma-checkin` |
| Compartilhar de qualquer app | **Compartilhar → Termux** |

Veja [TERMUX_SETUP.md](docs/TERMUX_SETUP.md) para o guia completo.

## Uma Ultima Palavra

> *"Ela pega o celular. E o silencio responde."*

Alguem que voce ama vai precisar da sua voz um dia e nao vai ter. Seu jeito de pensar. Seus erros. Sua fe. Sua sabedoria sobre amor, dor, dinheiro, Deus, relacionamentos, fracasso.
